# 问题 <span id="1">

用 ida 修改 demo.exe, 改变 demo.exe的运行。

```c
//demo.exe
//#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>


int main()
{
	int n;
	scanf("%d", &n);
	if (n > 0)
		printf("a > 0");  //IDA把'a'改成'n'
	else
		printf("n < 0");
}

```
